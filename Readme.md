# MetalicYoutubeToMp3

Downloader Youtube and change to Mp3

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install MyYoutubeToMp3.

```bash
pip install -r requirements.txt
```

## Usage

First add links to target.txt then run command below:

```python
python index.py
```
